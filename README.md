# LocalChatApp
    An application allowing users on the same network to communicate with eachother with the appication.

## Running the program
    Run MainProgram.java on Both Devices
    Use The Same Muticast IP Address (224.0.0.0 ~ 239.255.255.255) For Both Users
    Set The Same Port For Both Users
    Set Name for each User
    Select 'Enter'
    Type in message and press send
